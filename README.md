# 🛡️ CyberMap | Interactive Cybersecurity Domain Navigator

**CyberMap** is a high-performance, interactive mind map designed to navigate the complex landscape of cybersecurity domains. Built with **React** and **ReactFlow**, it provides a streamlined, minimalist interface for exploring specialized fields, from Architecture & Frameworks to Governance and Real-time Threat Scenarios.

## 🚀 Key Features

- **🌐 Interactive Navigation**: Explore cybersecurity domains through a dynamic, responsive mind map.
- **⚡ Real-time Threat Intelligence**: Simulated attack scenarios (Ransomware, Supply Chain, etc.) showing the propagation of threats across the network.
- **🇮🇹 Multi-language Support**: Full support for both **English** and **Italian** at the click of a button.
- **🔍 Deep Search & Filtering**: Instant discovery of concepts with automated path highlighting.
- **📤 HD Export**: Export the current map view in high resolution for documentation or presentations.
- **📱 Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

---

## 🛠️ Technology Stack

- **Framework**: [React 19](https://react.dev/)
- **Visuals**: [ReactFlow](https://reactflow.dev/) (Advanced nodes & custom edges)
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Build Tool**: [Vite 8](https://vitejs.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [MagicUI](https://magicui.design/)

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18+)
- npm / yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mahmoudabdel098/cyber-map.git
   ```
2. Navigate to the project folder:
   ```bash
   cd cyber-map
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 🎨 Design Philosophy

**CyberMap** follows a **"Cinematic & Technical"** aesthetic:
- **Clean Slate Background**: Minimale dot-grid background for focus.
- **Glassmorphism**: Translucent panels and sidebars for a premium feel.
- **Micro-Animations**: Subtle pulse effects on edges and spring-based node interactions.
- **High Contrast**: Strategic use of color (Rose-500 for threats, Blue-500 for paths) to guide the eye.
