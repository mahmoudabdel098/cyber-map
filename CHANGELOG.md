# Cyber Map Project Changelog

## [4.2.0] - 2024-02-28
### Added
- **Tight-Cluster Layout (ANTIGRAVITY_LAYOUT_GUARD)**: Implemented a new positioning logic that groups sub-nodes closely for a tidier look while maintaining strict non-overlap gaps.
- **Interactive Intelligence System**: Branch flow animations on edges and ghost titles in the background for group orientation.
- **Dual View Mode**: Added 'List View' for mobile optimization, accessible via a toggle.
- **Categorical Color Coding**: Each domain now has a unique vibrant color propagated through nodes, edges, and UI elements.
- **Node Enhancements**: Added data-stream ornaments, bit-meters, and dynamic shapes (circle, rect, diamond).
- **Technical UI**: Rulers, CAD-style corner accents, and a blueprint-style header.

### Changed
- **Mobile First Design**: Redesigned header and footer for compact screens.
- **Edge Routing**: Switched to 'step' type with directional pointers (ArrowClosed).
- **Layout Logic**: Migrated to a multi-layered ring system to handle high node density.
- **Zoom Dynamics**: Optimized min/max zoom and fitView padding for better navigation.

### Removed
- **Visual Noise**: Removed background stamps (Confidential, Top Secret) and security clearance badges to improve clarity.
- **Heavy Elements**: Simplified status bar by removing system load progress.
